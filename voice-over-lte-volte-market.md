<p><ul><li>Voice Over IP Multimedia Subsystem (VoIMS)</li><li>Circuit Switched Fall Back (CSFB)</li><li>Dual Radio/Simultaneous Voice and LTE (SVLTE)</li><li>Voice Over LTE Via Generic Access Network (VOLGA)</li><li>Single Radio Voice Call Continuity (SRVCC)</li></ul></p>
<p><p>Voice over LTE (VoLTE) is a technology that allows voice calls to be made over Long-Term Evolution (LTE) networks, which are typically used for data transmissions. There are different market types for VoLTE, such as Voice Over IP Multimedia Subsystem (VoIMS) where voice calls are transmitted using IP-based multimedia systems. Circuit Switched Fall Back (CSFB) allows calls to be switched to 3G or 2G networks if LTE is unavailable. Dual Radio/Simultaneous Voice and LTE (SVLTE) enables voice and data to be used simultaneously. Voice Over LTE Via Generic Access Network (VOLGA) enables voice calls over IP using Wi-Fi or other access networks. Single Radio Voice Call Continuity (SRVCC) allows smooth switchovers between LTE and 2G/3G networks during voice calls.</p></p>
